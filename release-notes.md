## What's New in v2.1.0

### New Features
- **Settings page** - Audio preferences, accent colors, Discord webhook, data management
- **Release checklist** - Seeded defaults (ISRC, mastering, artwork, etc.) with progress tracking; unlocked when song reaches 100%
- **Enhanced release blockers** - Color-coded severity (critical/warning/info) checking lyrics, audio files, tracking, visual direction
- **Collaborator notes** - Add notes per contributor with author names visible on each entry
- **Song report export** - One-click export a formatted text report of any song
- **Active review file selection** - Explicitly choose which audio file drives waveform per song
- **Audio upload from song page** - Upload button available directly in song detail view

### Bug Fixes
- **Task creation now works** - Fixed missing POST endpoint for song tasks/checklist
- **Fixed app icon** - Rebuilt from Forge Village logo

### Downloads
- **macOS (Apple Silicon)** - DMG installer or ZIP
- **Windows (x64)** - Portable .exe (no install required) or ZIP

### Notes
- macOS is signed for local development (no Developer ID certificate)
- Windows builds are portable - no installer needed
- Linux support coming soon
