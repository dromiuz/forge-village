About Forge Village

Version 1.0.0

Independent musician workspace — song-first, local-first. Built to help artists ship real work without fake SaaS sludge.

Features: Song tracking, lyrics, waveform markers, instrument tracking board, asset management, calendar, export/import.

Tech: Node.js + Express server, plain HTML/CSS/JS, local-first data storage.
